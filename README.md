# Slid to Do

- 사용자가 다양한 콘텐츠(아티클, 강의 영상, 줌 미팅 일정, 강의 PDF 등)를 할일 목록으로 관리
- 학습 진도나 프로젝트 진행 상황을 대시보드로 보여주며 각 할일에 대한 노트를 작성해 관리하는 서비스


### 기술 스택
- React, Typescript, Tanstack query, Tailwind, Storybook, Framer-motion
